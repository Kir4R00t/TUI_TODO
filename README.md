# TO-DO app made with TUI
Typical TO-DO app made using python with TUI front-end

## Running the app
### Windows
```python -m venv .venv``` <br>
```pip install -r requirements.txt``` <br>
```run.bat```
### Linux
```python3 -m venv .venv``` <br>
```pip3 install -r requirements.txt``` <br>
```run.sh```
